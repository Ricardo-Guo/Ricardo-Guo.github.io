AI Image Generator

A deep learning project for generating artistic images using AI

Jan 15, 2024

Overview

This project uses Generative Adversarial Networks (GANs) to create unique artistic images. The model was trained on over 10,000 artwork images from various styles and periods.

Technologies Used

  • Python - Main programming language
  • TensorFlow/Keras - Deep learning framework
  • OpenCV - Image processing
  • NumPy & Pandas - Data manipulation

Key Features

  • Generate high-resolution images (1024x1024)
  • Multiple style transfer options
  • Real-time preview
  • Export to various formats (PNG, JPG, WebP)
