Smart Home Automation

IoT-based home automation system with mobile app control

Mar 10, 2024

Introduction

A comprehensive smart home automation system that allows you to control lights, temperature, security cameras, and appliances from your smartphone or voice commands.

Hardware Components

  • Raspberry Pi 4 - Main controller
  • Arduino Uno - Sensor hub
  • ESP8266 - WiFi modules
  • Sensors: DHT11, PIR, Door/Window sensors
  • Relays - For appliance control
  • Camera - Motion detection

# main.py
from fastapi import FastAPI
import paho.mqtt.client as mqtt

app = FastAPI()

@app.post("/devices/{device_id}/control")
async def control_device(device_id: str, action: str):
    client = mqtt.Client()
    client.connect("localhost", 1883, 60)
    client.publish(f"home/devices/{device_id}", action)
    return {"status": "success", "device": device_id, "action": action}

# sensor_hub.py
import Adafruit_DHT
import time

SENSOR = Adafruit_DHT.DHT11
PIN = 4

def read_temperature():
    humidity, temperature = Adafruit_DHT.read_retry(SENSOR, PIN)
    return {
        'temperature': temperature,
        'humidity': humidity,
        'timestamp': time.time()
    }
